üöÄ Quick Start

1. Get Your Bot Token

Create a bot via @BotFather on Telegram

2. Configure Environment Variables

Set up required and optional variables:

  • BOT_TOKEN: Required - Your Telegram bot token
  • CWA_API_KEY: Optional - For earthquake data access
  • MCP_WEB_SEARCH_URL: Optional - Enhanced web search

3. Deploy

Choose your deployment method:

  • Vercel: One-click serverless deployment
  • Docker: Containerized deployment
  • Local: Development environment

4. Set Webhook

Configure the webhook URL and start using your bot!
